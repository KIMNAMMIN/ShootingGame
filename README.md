# ShootingGame
 space shooting과 여러가지 asset들을 활용해서 1인칭 시점 슈팅 게임을 만들 예정입니다. 타겟을 부수거나, 또는 목표지점에 슈팅을 하는 그런 게임이 될 것입니다.

단순하게 게임을 구현하는 것도 목표지만, 수업시간에 배웠던 기능을 이용해서 무언가 보여주고 싶습니다. 사실 현재까지 배운 것은 전반적인 rendering pipeline인데 그러한 그래픽 처리하는 것을 바로 안드로이드에서 어플으로 구현하는 것은 무리겠지만, 툴을 이용해서 어느 정도 특색을 갖춘 게임을 만들어볼 것입니다. 
 현재까지 배운 camera space, clip space등을 조금 더 현실적으로 보는 능력을 기르고 그 scene에서 타격 게임을 3d 기반으로 만들 것입니다.
